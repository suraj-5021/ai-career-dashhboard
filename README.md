# CareerOS Pro

AI-powered career development platform built with Next.js, TypeScript, Express.js, MongoDB, JWT Authentication, and Socket.IO.

## Features
- User Authentication
- Job Application Tracker
- Resume Analysis
- Interview Preparation
- Skills Roadmap
- Real-Time Notifications
- Admin Dashboard

## Tech Stack
- Next.js
- TypeScript
- Express.js
- MongoDB
- JWT
- Socket.IO

## Installation

Frontend:
npm install
npm run dev

Backend:
npm install
npm run dev
